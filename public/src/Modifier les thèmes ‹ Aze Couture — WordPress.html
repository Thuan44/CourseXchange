Forums d’entraide

Vous pouvez utiliser l’éditeur de thème pour modifier les fichiers CSS et PHP qui composent votre thème.

Commencez par choisir un thème à modifier via le menu déroulant, et cliquez sur « Sélectionner ». Une liste apparaîtra ensuite avec tous les fichiers modèles. En cliquant sur n’importe lequel de ces fichiers, vous verrez son contenu apparaître dans l’éditeur.

Pour les fichiers PHP, vous pouvez utiliser le menu déroulant « Documentation » pour en apprendre plus sur les fonctions trouvées dans le ce fichier. Le bouton « Consulter » vous enverra sur la page dédiée de cette fonction.

Lorsque l’on utilise un clavier pour naviguer :

  • Dans la zone de modification, la touche Tab saisit un caractère de tabulation.
  • Pour partir de cette zone, utiliser la touche ESC puis la touche TAB.
  • Personnes utilisant un lecteur d’écran : en mode formulaire, vous aurez besoin d’utiliser la touche ESC deux fois.

Après avoir fait vos modifications, cliquez sur « Mettre à jour le fichier ».

Conseil : gardez bien en tête que si vous modifiez le thème actuellement utilisé, vos visiteurs pourraient avoir une vision déroutante de votre site.

La mise à jour vers une nouvelle version du thème supprimera les modifications que vous aurez faites. Pour éviter cela, nous vous conseillons de passer plutôt par un Thème-Enfant.
